# ClickHouse + OpenTelemetry Monitoring

## Overview

This project demonstrates an observability pipeline using OpenTelemetry Collector and ClickHouse.

Telemetry data such as metrics, logs, and traces are collected, processed, and stored in ClickHouse for fast analytical queries and monitoring dashboards.
